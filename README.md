# NexCode - Coming Soon Page

This repository contains a simple **Coming Soon** static webpage for NexCode, hosted on **Cloudflare Pages** with the custom domain [nexcode.tn](https://nexcode.tn).

## 📁 Project Structure
├── index.html # Main coming soon page
├── assets/ # Static assets folder
│ └── logo # Logo image file
└── README.md # Project documentation

## 🚀 Deployment

The site is deployed automatically via **Cloudflare Pages**:

1. Connect this GitHub repository to Cloudflare Pages.
2. Set the build output directory to `/` (no build step required).
3. Cloudflare Pages will serve `index.html` as the entry point.
4. The custom domain `nexcode.tn` is configured in the Cloudflare Pages project settings.
